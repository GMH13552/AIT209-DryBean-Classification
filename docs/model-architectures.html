1. GCN (图卷积网络) — 从表格到图

核心创新：将无图结构的表格数据，通过 KNN 构造样本间图，再用消息传递进行节点分类。

2. KNN — 懒惰学习，近邻投票

不训练模型，直接存储全部训练数据。预测时计算距离，找 K 个最近邻居投票。

3. SVM — 最大间隔超平面 + RBF 核

寻找能最大间隔分隔各类别的超平面。RBF 核将数据映射到高维空间处理非线性。多分类采用一对一策略。

4. Random Forest — Bagging + 随机特征选择

Bootstrap 采样子集训练多棵决策树，每棵树在每个节点随机选择特征子集寻找最优分裂。最终由 100 棵树多数投票。

5. AdaBoost — 自适应样本加权

不拟合残差，而是动态调整样本权重：分错的样本权重 ↑，分对的 ↓。50 轮迭代后加权投票。
