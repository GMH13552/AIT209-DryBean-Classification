# 4. 算法实现 & 实验结果

## 4.1 算法选择

根据课程要求，实现了 **6 种多分类算法**，其中 3 种为课堂上没讲过、自行查阅资料实现的课外算法。

| # | 算法 | 类型 | 核心原理 | 来源 |
|:---:|------|------|------|:---:|
| 1 | KNN | 基于距离 | 多数投票，无训练 | 课堂 |
| 2 | SVM (RBF) | 核方法 | 最大间隔超平面 | 课堂 |
| 3 | Random Forest | 集成-Bagging | 多棵决策树投票 | 课堂 |
| 4 | **Gradient Boosting** | 集成-Boosting | 逐步拟合残差 | 课外 ✨ |
| 5 | **HistGradient Boosting** | 集成-Boosting | 直方图加速的 GBDT | 课外 ✨ |
| 6 | **MLP** | 神经网络 | 多层感知机+反向传播 | 课外 ✨ |

### 课外算法简介

**Gradient Boosting (GBDT)**
: 通过逐步添加弱学习器（通常为浅层决策树）来拟合前一步的残差。每步在负梯度方向上前进。是 XGBoost/LightGBM 等工业级框架的前身。

**HistGradient Boosting**
: sklearn 内置的高性能 GBDT 实现，灵感来自 LightGBM。将连续特征分箱为直方图，大幅加速训练。原生支持缺失值，无需手动填充。

**MLP (多层感知机)**
: 经典的前馈神经网络。使用 ReLU 激活、Adam 优化器、反向传播训练。虽然简单，但在结构化数据上常有不俗表现。

## 4.4 分析

### 为什么 SVM 表现最好？

1. RBF 核能捕捉特征间的非线性关系
2. 最大间隔原理天然防止过拟合（仅 0.95% Gap）
3. 标准化后 16 维中等特征空间很适合 SVM
4. 可以用较小的 C=10 平衡间隔和误差

### 为什么 Boosting 过拟合？

1. GBDT 和 HistGBDT 在训练集上达到 **100%** 精度——完全记住了训练数据
2. max_depth=6 的树仍然太深，需要降低或增加正则化
3. 改进方向：降低 max_depth → 3，增加 subsample，使用 early_stopping

### 速度对比启示

- **MLP 推理最快**（1.9ms）：矩阵乘法硬件加速
- **KNN 推理最慢**（480ms）：需要计算所有训练样本距离
- **实际部署**：优先考虑 MLP 或 Random Forest 树模型
